# 🌍 Trewel — В каких странах ты был?

Премиальная мобильная веб-форма для Telegram-групп. Пользователь выбирает
страны, в которых побывал, отправляет ответ — и его список появляется в
**общей публичной ленте** путешествий группы.

Дизайн в стиле **spatial web / dark-green luxury**: глассморфизм, мягкий
неоморфизм, кинематографичные переходы и «живые» карточки стран с
анимациями выбора (Framer Motion).

## ✨ Возможности

- **Лендинг** с парящими 3D-объектами и cinematic-входом
- **Шаг с именем / Telegram-юзернеймом** (неоморфный инпут)
- **Выбор стран** — 90+ стран, поиск, фильтр по регионам, премиальные карточки
- **Анимация выбора** — spring-чекмарк, свечение, световой свип, акцент страны
- **Модалка подтверждения** перед публикацией
- **Публичная лента** «Карта путешествий группы»:
  - Все участники
  - 🔥 Самые популярные страны
  - 🏆 Кто посетил больше всего стран
  - 💎 Редкие страны
  - Анимированные счётчики
- **Шеринг в Telegram** одной кнопкой
- Работает как **Telegram Mini App** (авто-разворот, haptics, prefill имени)

## 🧱 Стек

Next.js 14 (App Router) · TypeScript · Tailwind CSS · Framer Motion · Vercel KV

## 🚀 Запуск локально

```bash
npm install
npm run dev
```

> Локально данные хранятся в памяти процесса (`next dev`) — приложение сразу
> работает без всякой настройки. Общая лента для всех включается на Vercel KV.

## 🗄️ Хранилище — Vercel KV (без отдельной БД)

Общие результаты группы хранятся в **Vercel KV** через API-роут
`app/api/submissions/route.ts`. Никакого SQL и отдельной регистрации.

1. Vercel Dashboard → **Storage → Create → KV** → назови, например `trewel-kv`
2. **Connect to Project** → выбери проект
3. Переменные `KV_REST_API_URL` и `KV_REST_API_TOKEN` добавятся **автоматически**

Если KV не подключён, API отдаёт пустой список (в проде) либо in-memory
(локально), а клиент в крайнем случае падает на `localStorage`.

## ☁️ Деплой на Vercel

1. Импортируй репозиторий на [vercel.com](https://vercel.com), Root Directory → `travel-app`
2. Подключи KV (см. выше) — переменные добавятся сами
3. (Опц.) добавь `NEXT_PUBLIC_APP_URL` для кнопки шеринга
4. Deploy → получишь публичный URL

## 🤖 Telegram Mini App

1. @BotFather → создай бота → **Mini Apps → Enable** → вставь URL с Vercel
2. (Опционально) запусти `bot/bot.mjs` с `BOT_TOKEN` и `WEBAPP_URL`, чтобы
   `/start` открывал приложение кнопкой.

## 📁 Структура

```
app/            — страницы (flow + /results) + api/submissions (Vercel KV)
components/     — переиспользуемые компоненты (см. список в задаче)
lib/            — данные стран, типы, api-клиент, статистика
bot/            — Telegram-бот (long polling)
```
